## 📋 О проекте
Умный помощник для пациентов и врачей. Автоматизирует ведение медицинской карты, напоминания о лекарствах, распознавание анализов и проверку безопасности лечения.

## ✨ Возможности
- ✅ Голосовой ввод жалоб и назначений
- ✅ Напоминания о лекарствах
- ✅ Распознавание анализов по фото
- ✅ Проверка лекарственных взаимодействий
- ✅ Личный кабинет врача
- ✅ Панель управления клиникой
