# Медицинский бот для MAX

## Описание
Умный помощник для пациентов и врачей. Автоматизирует ведение медицинской карты, напоминания о лекарствах, распознавание анализов и проверку безопасности лечения.

## Возможности
- 🎤 Голосовой ввод жалоб
- 💊 Напоминания о лекарствах
- 📄 Распознавание анализов по фото
- 🔐 Проверка полипрагмазии (Beers, STOPP)
- 👨‍⚕️ Личный кабинет врача
- 🏥 Дашборд для клиник
- 💳 Подписки и платежи
- 🔗 Интеграция с МИС (MEDODS, 1С)

## Технологии
- **Языки:** Go, Python
- **Базы данных:** PostgreSQL, Redis
- **Очереди:** RabbitMQ
- **Контейнеры:** Docker, Docker Compose
- **CI/CD:** GitHub Actions
